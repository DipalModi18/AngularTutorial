# FormApplication2

Application starts with a login form.
Upon successful login, Application redirects to profile page just showing a welcome message.
Both Username and password are required fields.
User not having an account can register.
Temporarily, registered user data is not saved anywhere.
For registration in-built as well as custom validations are used.
The Form Application uses Reactive Form.
